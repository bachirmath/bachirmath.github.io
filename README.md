---------------

لتكن $f$ دالة، يسمى
$$\mathscr{L}(f)(s):=\int_{0}^{+\infty}f(x)e^{-sx}dx$$ (في حالة تقارب
التكامل)، **[بتحويل لابلاس]{.underline}** ل $f$.
